About

This is a Spotify Clone, a music player web app inspired by the real Spotify. It is built using HTML, CSS, and vanilla JavaScript as a learning project to understand the fundamentals of web development.

This project was built with the help of Code With Harry, a popular programming tutorial channel. It includes core features like playing songs, a seekbar, previous and next controls, and live album art display.

Disclaimer: This project is made purely for educational purposes. All song names, album covers, and artist names belong to their respective owners. This is not affiliated with or endorsed by Spotify in any way.
